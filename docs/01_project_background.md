# 01 Project Background

## 1. 项目背景

地下矿房开采后，原本由矿体承担的应力会重新分配，矿房周围围岩可能发生破坏、松动、崩落和塌陷。对于这类问题，单纯连续介质方法或单纯离散元方法各有局限：

- 连续介质方法适合描述远场围岩应力分布和整体变形；
- 离散元方法适合描述近场围岩破裂、块体运动、崩落和大变形过程。

因此，本项目采用 **PFC2D 6.0 与 FLAC2D wall-zone 耦合** 的方式，将近场离散颗粒区与远场连续介质区结合起来，模拟地下矿房采空后的围岩崩落塌陷过程。

## 2. 工程问题简化

当前阶段的主要任务是打通整体数值模拟流程，而不是立即建立复杂工程模型。

因此，模型初期可以简化为：

- 单种岩石；
- 矩形矿房；
- 二维模型；
- 通过删除 PFC 颗粒模拟矿房开采；
- FLAC zone 区域先采用原始版本中的本构模型和参数设置；
- 重点验证 PFC-zone 耦合、原岩应力恢复、采空区形成和后续计算流程是否连通。

在流程打通之后，再逐步引入更复杂的工程特征。

## 3. 后续工程化扩展方向

流程稳定后，模型计划扩展为更接近真实环境的形式：

1. **倾斜矿房形状导入**：通过 DXF 文件导入倾斜矿房形状，不再局限于矩形矿房。
2. **多区域岩性赋参**：通过导入多个 DXF 区域，对不同区域的 PFC 颗粒进行分组和赋参，从而模拟不同岩石或不同工程地质区域。
3. **更真实的采场结构**：逐步加入复杂矿体形态、围岩分区、裂隙或弱面等因素。

## 4. 方法定位

本项目不是单纯的 PFC 颗粒堆积模拟，也不是单纯的 FLAC 连续介质模型，而是：

> 基于 PFC2D-FLAC2D wall-zone 耦合的地下矿房围岩崩落塌陷模拟。

其中：

- PFC2D 区域用于描述矿房附近的离散破坏与崩落；
- FLAC2D zone 区域用于描述远区围岩连续应力场；
- wall-zone 耦合边界用于在二者之间传递力学响应。

## 5. 当前核心矛盾

原始版本已经实现了可运行的耦合流程，但 PFC 颗粒部分未充分恢复原岩应力状态。

这导致后期计算中可能出现：

- 颗粒初始应力不合理；
- 上部颗粒离开耦合墙；
- 采空后崩落响应受初始状态影响较大；
- PFC 与 FLAC 的耦合边界受力状态不够可靠。

因此，当前阶段的核心任务是：

> 在保留原始版本耦合流程和自动保存逻辑的基础上，重构 PFC2D 颗粒建模流程，通过伺服墙体、颗粒赋参和接触模型设置恢复原岩应力。
