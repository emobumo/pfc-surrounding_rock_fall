# PFC2D-FLAC2D Stope Caving Simulation

本仓库用于保存一个基于 **PFC2D 6.0 与 FLAC2D wall-zone 耦合** 的地下矿房围岩崩落塌陷模拟项目。

## 项目目标

本项目旨在通过 PFC2D 与 FLAC2D 的耦合建模，模拟地下矿房采空后围岩的破坏、崩落、塌陷及远区应力响应。

当前阶段的核心任务是优先 **打通整体模拟流程**，而不是立即建立复杂工程模型。当前可简化为单种岩石、矩形矿房；流程稳定后，再通过 DXF 导入倾斜矿房形状，并通过多个 DXF 区域完成不同岩性区域的颗粒分组与赋参。

## 软件与方法

- 软件：PFC2D 6.0 + FLAC2D
- 耦合方式：wall-zone 耦合
- PFC 区域：矿体及矿房附近围岩，负责模拟破坏、崩落、大变形和颗粒运动
- FLAC zone 区域：包裹 PFC 区域的远场围岩，用于施加约束并观察远区应力分布影响
- 开挖方式：删除矿房对应区域的 PFC 颗粒形成采空区

## 当前模型理解

整体模型为 **zone 区包裹 PFC 区域**：

```text
FLAC2D zone 远场围岩
┌──────────────────────────────┐
│                              │
│ PFC2D 颗粒区域 矿体 + 近场围岩 │
│   ┌──────────────────────┐   │
│   │                      │   │
│   │ 矿房区域：删除颗粒    │   │
│   └──────────────────────┘   │
│                              │
└──────────────────────────────┘
```

## 原始版本价值

原始版本已经验证可以运行，最重要的价值是：

1. 已打通 PFC-zone 耦合流程；
2. 已包含开挖后模型状态自动保存逻辑；
3. 提供了“颗粒生成—耦合—开挖—保存”的参考流程。

原始版本的主要问题是：PFC 颗粒部分未充分恢复原岩应力，后期可能出现上部颗粒离开耦合墙的现象。因此后续需要重构 PFC 初始建模和原岩应力恢复流程。

## 当前改写方向

PFC 部分改写主要参考本地案例：**含裂隙岩体原岩应力平衡与开挖扰动模拟**。该案例为 PFC5.0 版本，且本地尚未运行验证，因此需要先完成：

1. PFC5.0 到 PFC2D 6.0 的语法转换；
2. 原案例建模逻辑梳理；
3. 伺服墙体、颗粒赋参、接触模型设置流程的提取；
4. 伺服恢复原岩应力后的耦合墙与 zone 区域尺寸重新对接。

## 当前阶段

- 已完成部分 PFC 改写；

- 后续根据更多代码案例，可能继续添加新的功能模块。
